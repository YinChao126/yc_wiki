# 谷歌云搭建SSR服务

作者：尹超

日期：2019-6-28

背景：科学上网哪家强，谷歌免费VPN（SSR）帮你忙



## 大致思路

- 先申请免费的谷歌云GCP
- 然后创建防火墙规则
- 再创建VM计算实例
- 再创建静态IP并绑定实例
- 登陆VM实例并进行SSH配置
- 根据教程实现SSR配置

## 详细步骤

### 申请GCP，再激活

### 新建防火墙规则

重点参数如下：

```
名称：boom
方向：入站
目标为:网络中的所有实例
对匹配项执行的操作：允许
IP范围： 0.0.0.0/0
协议和端口：全部允许
```

### 创建VM实例

注意勾选防火墙的两个允许：（允许HTTP流量，允许HTTPS流量）

点开网络，在网络标记处填入防火墙的名字:boom

### 创建静态IP

记得与VM实例绑定

### 登陆VM实例，进行SSH配置

输入(原样输入即可)：
`bash <(curl -s -L https://233blog.com/ss.sh)`

```
端口号：32617
密码：123456789
加密协议：chacha20-ietf-poly1305
```
