# Linux更新源

## keywords

linux,  source, update

## 1.先备份

`sudo cp /etc/apt/sources.list /etc/apt/sources.list.bck`

## 2.修改/etc/apt/sources.list文件

完整拷贝如下任意一个源即可（全部覆盖）

`sudo vim /etc/apt/sources.list`
